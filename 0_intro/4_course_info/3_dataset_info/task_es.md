Muchas de las tareas del curso se basan en un único conjunto de datos que utilizarás a lo largo de los módulos.

Este conjunto de datos contiene información detallada sobre las ventas y calificaciones de videojuegos. Fue compilado por [Rush Kirubi](https://www.kaggle.com/rush4ratio) en 2016, con datos recopilados de [VGChartz](https://www.vgchartz.com/) y [Metacritic](https://www.metacritic.com/). Incluye ventas globales y regionales, géneros de juegos, plataformas, editores, así como calificaciones de críticos y usuarios.

Puedes explorar el conjunto de datos directamente en Kaggle usando [este](https://www.kaggle.com/datasets/rush4ratio/video-game-sales-with-ratings/data) enlace.

A lo largo del curso, usarás este conjunto de datos para crear visualizaciones significativas y descubrir patrones interesantes, como qué plataformas dominaron las ventas en diferentes regiones, cómo las calificaciones se relacionan con las ventas y cómo cambiaron las tendencias con el tiempo.

Como se mencionó anteriormente, este curso se centra en la visualización de datos, por lo que no cubriremos el procesamiento de datos. En su lugar, utilizarás funciones ocultas que convierten automáticamente el conjunto de datos al formato requerido.  
Si lo deseas, aún puedes visualizar el conjunto de datos resultante — ya sea imprimiéndolo con la función básica `print` o inspeccionándolo con el depurador integrado del IDE.
