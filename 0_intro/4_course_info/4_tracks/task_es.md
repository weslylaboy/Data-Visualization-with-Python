En este curso, utilizamos dos bibliotecas para la visualización de datos: [Seaborn](https://seaborn.pydata.org/index.html) y [Matplotlib](https://matplotlib.org/).  
Cada una tiene su propia ruta de aprendizaje dedicada.

Si eres nuevo en la visualización de datos, te recomendamos comenzar con Seaborn.  
Ofrece una API más sencilla y te permite crear gráficos atractivos con un esfuerzo mínimo.  
Sin embargo, sus opciones de personalización son limitadas, lo que puede ser una restricción para visualizaciones más avanzadas.

Matplotlib, por otro lado, es una biblioteca más poderosa y flexible.  
Te permite crear prácticamente cualquier tipo de gráfico con un control detallado sobre cada elemento.  
Dicho esto, esta flexibilidad viene acompañada de una curva de aprendizaje más pronunciada, lo que significa que necesitarás invertir más tiempo para lograr los resultados deseados.

Aunque las dos rutas de aprendizaje están diseñadas para ser independientes y pueden completarse en cualquier orden,  
te aconsejamos no realizarlas simultáneamente.  
Enfocarte en una biblioteca a la vez te ayudará a construir una comprensión más clara y a evitar confusiones entre las dos APIs.
