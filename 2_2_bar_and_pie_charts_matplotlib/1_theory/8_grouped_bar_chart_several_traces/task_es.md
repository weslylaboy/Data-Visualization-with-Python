## Teoría

Hasta ahora, solo hemos graficado las ventas de una región, ¡así que agreguemos más!

## Tarea

Usa la función oculta `get_all_regions`, que devuelve todas las regiones de ventas.  
Itera sobre el resultado y grafica las ventas de cada región utilizando la función `plot_region`.

Si lo prefieres, puedes obtener todas las regiones manualmente. Consulta las pistas correspondientes a continuación.

## Pistas

<div class="hint" title="¿Cómo obtener todas las regiones?">
  Para obtener todas las regiones, puedes usar el método <a href="https://pandas.pydata.org/docs/reference/api/pandas.Series.unique.html"><code>unique</code></a> en la columna <code>region</code>.
