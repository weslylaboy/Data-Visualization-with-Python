Los histogramas son ideales para visualizar la distribución de datos continuos unidimensionales.  
Dividen el rango de valores en intervalos,  
lo que facilita identificar agrupaciones, asimetrías y valores atípicos.  
Esto los hace especialmente útiles para grandes conjuntos de datos,  
proporcionando una visión general rápida sin necesidad de inspeccionar cada punto de datos.

Los diagramas de dispersión también pueden mostrar la distribución, particularmente en 2D o 3D.  
Sin embargo, para datos unidimensionales, son menos efectivos para mostrar la densidad, aunque son útiles para identificar valores atípicos.  
Reducir la opacidad de los puntos ayuda a aproximar la densidad: las áreas más oscuras indican mayor solapamiento.  
Por eso los diagramas de dispersión suelen complementar a los histogramas en análisis más profundos.

Cuando a Jonsi le ofrecieron un ascenso, su tarea parecía sencilla:  
analizar datos de lluvias en la ciudad, cruciales para planificar mejoras en el sistema de drenaje y prevenir inundaciones.  
Inicialmente, creó un diagrama de dispersión para examinar la distribución.  
Para abordar el problema de que los datos de precipitación se colapsaban en una sola línea, Jonsi añadió un poco de "jitter".  
Esto reveló un valor atípico y una región donde los datos parecían bastante uniformemente distribuidos,  
lo cual le pareció inusual.  
Recordó entonces que los histogramas son más adecuados para este tipo de análisis.  
Después de trazar un histograma, se dio cuenta de que los datos no eran uniformes en absoluto: se agrupaban alrededor de niveles específicos de lluvia.  
Este hallazgo expuso los umbrales críticos donde el sistema de drenaje fallaría.
